Gem Collector

Some instructions to play the game


  1. At the start of the game there will be a random number to beat."Number To Match"
  2. Below there are four gems.
  3. Clicking on each gem will give a specific number, accumulate the number to beat the random number provided by the game.
  4. You win the game by matching your total score to the random number.
  5. If your score goes above the random number you lose.
  6. The value of each crystal is random.
  7. The gems value changes on each game.
